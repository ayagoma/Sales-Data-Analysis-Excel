Sales Data Analysis (Excel Dashboard)

This project focuses on analyzing sales data using Microsoft Excel.

Tools Used:
- Microsoft Excel

Key Tasks:
- Data cleaning and preparation
- Creating pivot tables
- KPI analysis
- Building an interactive dashboard

Dashboard Includes:
- Total sales and revenue KPIs
- Monthly sales trends
- Top-performing products
- Sales by region

Outcome:
- Provided insights to support business decision-making
